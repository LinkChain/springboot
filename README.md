## springboot项目说明
- springboot项目是初学Spring Boot的学习和练习springboot的源码。Spring Boot是用来简化Spring应用初始搭建以及开发过程的全新框架,SpringBoot是SpringMVC升级版，使得编码、配置、部署、监控都将变得更简单。
## SpringBoot的特点：
- 化繁为简，简化配置
- 备受关注，是下一代框架
- 微服务的入门级微框架
